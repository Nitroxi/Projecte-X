# Projecte X

Aquest repositori està pensat per guardar activitats, pràctiques i treballs que no entren dins de cap projecte concret.

La idea de **Projecte X** és tindre un espai general on reunir tot aquest material de manera ordenada, sense barrejar-lo amb altres repositoris més específics.

![img](https://media4.giphy.com/media/v1.Y2lkPTc5MGI3NjExdXMxeHMwbHFmdGJ3MnZxa2Exemw5MGxhYnp5b2xtd3YxMjA0ZnoxNyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/iK9JiJeRBypzJpZWEb/giphy.gif)

## Objectiu

L'objectiu d'aquest repositori és centralitzar activitats soltes en un únic lloc per mantindre una millor organització del treball.

## Nota

Aquest repositori s'anirà actualitzant a mesura que s'afegisquen noves activitats.
